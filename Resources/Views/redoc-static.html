v1 API

週刊番組表を取得

1週間分全ての番組情報を返却する。 実行日の日本時間の0時から7日後の23時59分までの間の番組情報を返却する。

Responses

production

https://agqr.sun-yryr.com/api/all

local

http://localhost:8080/api/all

Response samples

  • 200
Content type
application/json
[
  • {
    • "title": "string",
    • "ft": "2019-08-24T14:15:22Z",
    • "to": "2019-08-24T14:15:22Z",
    • "pfm": "string",
    • "dur": 0,
    • "isBroadcast": true,
    • "isRepeat": true
    }
]

日間番組表を取得

本日放送予定の番組情報を返却する。 本日とは、日本時間の0時から23時59分までの間のことを指す。

production

https://agqr.sun-yryr.com/api/today
